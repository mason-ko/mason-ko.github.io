## 1. overcommit_memory

Host PC에서 실행

```
sudo sysctl vm.overcommit_memory=1

#재부팅시 적용되려면
vi /etc/sysctl.conf
# 추가
net.core.somaxconn=1024
```



## 2. THP Disable

Host PC에서 실행

```
echo never > /sys/kernel/mm/transparent_hugepage/enabled

#재부팅시 적용되려면
vi /etc/rc.local
# 추가
echo never > /sys/kernel/mm/transparent_hugepage/enabled
exit 0
```


## 3. TCP Backlog

Docker Compose yaml에 내용 추가 ( 65535 까지 가능 )  
sysctls 은 docker compose version 2.1 이상부터 가능함을 확인

```
sysctls:
  - net.core.somaxconn=1024
```
